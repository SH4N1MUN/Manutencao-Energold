# ENERGOLD PWA para GitHub Pages

## Arquivos
- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icons/`

## Como publicar no GitHub Pages
1. Crie um repositório ou use o seu atual.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. No GitHub, vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (ou `master`)
   - **Folder:** `/root`
5. Salve.
6. Aguarde o link do Pages ficar disponível.

## Como instalar no iPhone
1. Abra o link publicado no **Safari**.
2. Toque em **Compartilhar**.
3. Toque em **Adicionar à Tela de Início**.
4. Abra pelo ícone criado.

## Observações
- O app abre offline com os arquivos principais em cache.
- A sincronização com Google Sheets depende de internet.
- Sempre abra pelo link do GitHub Pages, não pelo app Arquivos.
