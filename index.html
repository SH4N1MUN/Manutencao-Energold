Integração SharePoint Microsoft 365
🔐 Autenticação Microsoft
Como funciona:
O app usa a API Microsoft Graph para salvar cada OS como uma linha no SharePoint. É gratuito para qualquer tenant Microsoft 365. Basta registrar o app no Azure AD e configurar abaixo.
Status: aguardando configuração
📖 Guia de Configuração
1
Acesse portal.azure.com → Azure Active Directory → Registros de Aplicativos → Novo Registro
2
Nome: Energold OS Manager — Tipo de conta: Esta organização — URI de redirecionamento: SPA → http://localhost + URL onde hospedar
3
Em API Permissions, adicione: Sites.ReadWrite.All (delegated) e conceda consentimento de administrador
4
Copie o ID do Aplicativo (cliente) e o ID do Diretório (locatário) e cole nos campos acima
5
No SharePoint, crie uma Lista chamada OS Lançadas com os campos correspondentes (ou use o botão Criar Lista abaixo)
